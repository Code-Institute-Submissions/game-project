Instructions:

This game uses keys to position the block on screen.

The controls are as follows

  • Move Left: "A" key
  • Move Right: "D" key
  • Drop Down: "S" key
  • Rotate Right: "E" key
  • Rotate Left: "Q" key

Alternatively, if you are on a mobile or tablet device, you can use the keypad located below the game.
