Instructions:

This game uses keys to position the blocks on screen.

The controls are as follows

  • Move Left: "A" key
  • Move Right: "D" key
  • Drop Down: "S" key
  • Rotate Right: "E" key
  • Rotate Left: "Q" key

Alternatively, if you are on a mobile or tablet device,
you can use the keypad located below the game.

Although this game is created to cater for all devices and screen sizes, the optimal setup is a screen width between 1024px & 1440px and using a desktop or laptop computer.

This game is tested on multiple devices and browser types, should you come across an aspect of the game that you think can be improved, please let me know using the contact form, which can be accessed from the main menu.
